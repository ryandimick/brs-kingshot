## How to read this document

- Each pack has one or more **availability windows** within the 28-day cycle.
- A "window" is a contiguous run of days during which the pack is offered. Adjacent days share inventory — you do **not** get a refreshed supply each day within the same window.
- Some packs appear in multiple non-contiguous windows within the same cycle (e.g., Charm Craftsman, Custom Arms Set, Conqueror). Each window is listed separately.
- One window crosses the cycle boundary (Combat Medic, Days 28→1).
- Pack contents, pricing tiers, and purchase recommendations are tracked in **Phase 2** (not yet populated).

### Schema notes

- **Pack label** is the in-game tag shown on the pack tile. Two formats observed so far:
  - **Percentage labels** (e.g., Conqueror "2090%", Custom Arms Set "1800%") — the game's stated value vs. raw gem cost at that price point. Not an independent valuation.
  - **Categorical labels** (e.g., Hope Market "Best Deals", Custom Forging Set "Best Deals") — qualitative tags applied across an entire pack family rather than per-tier.
  Captured per-tier (or per-pack-family if uniform) where observed.
  
  **Label format does not strictly correlate with tier-access model.** Both independent-access packs (Custom Arms Set: percentage; Hope Market: categorical) and sequential-unlock packs (Conqueror: percentage) appear in mixed combinations. The dimension determining label format is not yet identified.
- **Percentage label behavior** (when used) differs by tier-access model:
  - **Independent-access packs** with percentage labels (Custom Arms Set, Custom Pet Chest): percentage is **constant across all tiers**.
  - **Sequential-unlock packs** with percentage labels (Conqueror, Exclusive Gear Enhancement Special, Truegold Value Pack): percentage **decreases as tier price increases**.
- **Resource entries** are recorded as `unit count × pack size` (e.g., `400 × 10K Bread`) with implied total in parentheses.
- **Choice mechanic** describes any selection step (e.g., "pick 1 of 4") if applicable; "Fixed bundle" if not.
- **Tier-access model** describes how higher tiers are unlocked. Two models observed:
  - **Independent access** — all tiers visible and purchasable in any order (Hope Market, Custom Arms Set, Custom Pet Chest, Custom Forging Set).
  - **Sequential unlock** — higher tiers locked behind purchase of prior tier (Conqueror, Exclusive Gear Enhancement Special).
- **Tier naming patterns** correlate with tier-access model:
  - **Independent-access packs** typically name each tier (e.g., "Common Custom Chest," "Kindling Light Pack").
  - **Sequential-unlock packs** typically use only a pack-family name with no per-tier names (e.g., "Conqueror" $4.99 tier, "Exclusive Gear Enhancement Special" $4.99 tier).
- **Per-window cap.** Each tier of each pack can be purchased **1 time per window** (default, per Ryan's observation across all documented packs and tiers). Per-pack overrides will be noted at the pack level if different behavior is observed.
- **Source** indicates how the data was captured. `In-game (Ryan)` = directly observed by Ryan. Web sources noted with date and URL.

#### Hope Market — Structural notes

- **Pack label:** "Best Deals" (categorical) — applied uniformly across all 5 tiers, not per-tier.
- **Tiers are independently accessible.** Players are *not* required to purchase lower tiers before unlocking higher ones. Each tier can be purchased directly based on budget.
- **Linear scaling.** Across all five tiers, contents scale exactly proportionally with price:

  | Tier | Price | Multiplier vs. $4.99 baseline |
  |---|---|---|
  | Kindling Light | $4.99 | 1× |
  | Midnight's Hope | $9.99 | 2× (price 2.00×) |
  | Moonlit Aspiration | $19.99 | 4× (price 4.01×) |
  | Morning Star's Vow | $49.99 | 10× (price 10.02×) |
  | Morning Glow | $99.99 | 20× (price 20.04×) |

  $/value is identical at every tier within rounding.
- **Choice mechanic preserved across all tiers.** Same 4 options (Truegold, Governor Gear, Artisan's Vision, Hero Gear) at every price point. The bonus content scales linearly within each option.
- **Per-purchase choice scope.** Each purchased pack triggers one selection of 1-of-4. Buying multiple tiers in the same window allows multiple selections (each independent).
